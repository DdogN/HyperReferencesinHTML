What the code does

a href

What the a does in a href is set an anchor setting where you are coming from basiccly. Then the href (said as Hyper Reference) is telling you where you want to go (in my case google). There is a problem with some websites like google. the problem is you have to put https:// in front of it, this is because it is a secure website and is only for certain sites.

="">

What these things do is tell your computer what URL you want to go to, the = is telling the hyper reference that there is something to go to. The quotes are telling the computer where the url is. The end bracket is saying that the URL is done and that the content is about to start

/a

The /a tag is the simplest tag there is, it is just ending the anchor tag.

Button (optional)

The button code is one of the coolest tags in HTML, it is a simple tag that creates a small button. You can make the button more complicated with CSS and HTML. It is very cool and it overlaps the link function making it all one button.
